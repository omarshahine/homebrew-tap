# omarshahine/homebrew-tap

Homebrew tap for personal CLIs.

## Install

```bash
brew tap omarshahine/tap
brew install <formula>
```

## Formulas

| Name | Description |
|------|-------------|
| [`aqara-cli`](Formula/aqara-cli.rb) | Aqara Open Cloud API client — homes, devices, rooms, scenes, rename/move, OAuth bootstrap, launchd refresher |
| [`findmy-cli`](Formula/findmy-cli.rb) | Query Find My friend locations on macOS via UI scraping |
| [`lutron-cli`](Formula/lutron-cli.rb) | Lutron Caseta Smart Bridge client — devices, scenes, area rename/create/delete, device move |
